# License Plate Detection and Recognition System

## Project Overview
This project detects and recognizes vehicle license plates from video using Python and Computer Vision.

## Technologies Used
- Python
- OpenCV
- Tesseract OCR
- NumPy

## Features
- Detect moving vehicles
- Find license plate region
- Extract number using OCR
- Display detected plate numbers
- Save screenshots

## Modules

1. Video Input Module
2. Vehicle Detection Module
3. License Plate Detection Module
4. OCR Recognition Module
5. Result Display Module

## How to Run

Install libraries:

pip install -r requirements.txt

Run:

python license_plate_detector.py
